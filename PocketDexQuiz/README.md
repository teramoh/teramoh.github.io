# 🎮 ポケモン分類クイズ - PocketDex Quiz

ポケモンの「ぶんるい」を当てるクイズアプリです。

## 🌐 デモ

https://teramoh.github.io/PocketDexQuiz/dist

## 📖 概要

ポケモン図鑑に記載されている「ぶんるい」（例：ねずみポケモン、たねポケモンなど）を4択クイズ形式で出題します。

### 機能

- ✅ ランダムなポケモンが出題される
- ✅ 4択形式で「ぶんるい」を当てる
- ✅ 問題数を5/10/15/20問から選択可能
- ✅ 正解・不正解のフィードバック
- ✅ スコアとプログレス表示
- ✅ 結果画面で成績に応じたメッセージ表示
- ✅ レスポンシブデザイン（スマートフォン対応）

## 🛠️ 技術スタック

- Vite
- Vanilla JavaScript
- CSS3（カスタムプロパティ、グリッド、アニメーション）

## 📁 プロジェクト構造

```
PocketDexQuiz/
├── src/
│   ├── data/
│   │   └── pokemon.js    # ポケモンデータ
│   ├── main.js           # メインアプリケーション
│   └── style.css         # スタイル
├── dist/                 # ビルド出力（公開用）
├── index.html
├── vite.config.js
└── package.json
```

## 🚀 開発

### 必要環境

- Node.js 20.19+ または 22.12+

### セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# 本番ビルド
npm run build

# ビルドのプレビュー
npm run preview
```

## 📜 データソース

ポケモンの分類情報は[ポケモンずかん公式サイト](https://zukan.pokemon.co.jp/)を参考にしています。

## ⚠️ 注意事項

- 現在のデータは第1世代（カントー地方）のポケモン151匹に対応
- 画像は公式図鑑から読み込んでいます
